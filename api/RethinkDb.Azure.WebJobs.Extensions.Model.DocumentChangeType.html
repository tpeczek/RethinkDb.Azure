Uninitial

If an initial result for a document has been sent and a change is made to that document that would move it to the unsent part of the result set (e.g., a changefeed monitors the top 100 posters, the first 50 have been sent, and poster 48 has become poster 52), an 'uninitial' notification will be sent, with an old_val field but no new_val field.
